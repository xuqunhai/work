DOM事件
    基本概念：DOM事件级别
    DOM0: ele.onclick
    DOM2: eledocument.addEventListener('click', (event) => {});
    DOM3: eledocument.addEventListener('keyup/鼠标/键盘事件', (event) => {});
    
    DOM事件模型：捕获/冒泡
    DOM事件流：捕获-目标阶段-冒泡
    描述DOM事件捕获的具体流程
    window - document - html - body - ... - 目标元素
    Event对象应用
    event.preventDefault/stopPropagation/currentTarget/target/stopImmediatePropagation()
    target - 点击对象
    currentTarget - 事件绑定元素
    stopImmediatePropagation - 事件优先级，A->B，A执行后不执行B可用
    自定义事件
    var eve=new Event('haha')
    ele.addEventListener('haha', function(){});
    ele.dispatchEvent(eve)
    CustomEvent比Event优点在于还能接受参数
