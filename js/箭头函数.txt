箭头函数表达式的语法比函数表达式更短，
并且没有自己的 this，arguments，super 或 new.target。

这些函数表达式更适用于那些本来需要匿名函数的地方，并且它们不能用作构造函数。

语法为:

(参数 1, 参数 2, …, 参数 N) => {函数声明}

// 相当于：(参数 1, 参数 2, …, 参数 N) => {return 表达式;}
(参数 1, 参数 2, …, 参数 N) => 表达式（单一）

// 当只有一个参数时，圆括号是可选的
(单一参数) => {函数声明}
单一参数 => {函数声明}

// 没有参数的函数应该写成一对圆括号
() => {函数声明}
