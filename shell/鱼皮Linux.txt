ifconfig
查看网卡

free -h
检查服务器下内存是否不够，显示available就是剩下可用内存

df -lh
查看磁盘空间是否足够

du -sh *
进入某项目根路径后执行，可查看项目占用多少空间

cat read.md
查看文件

查看系统版本
uname -a

centos自带yum安装java：
yum install java-1.8.0-openjdk* -y

查看java版本号
java -version

查看java安装目录
which java

安装maven构建java项目，使用wget下载压缩包
wget https://mirror-hk.koddos.net/apache/maven/maven-3.8.2/binaries/apache-maven-3.8.2-bin.tar.gz

用tar命令解压
tar -zxvf xxx.tar.gz

构建命令
./apache-maven-3.8.2/bin/mvn install

查看生成jar包在哪
find -name '*.jar*'

把jar包复制到外面目录，方便查找
cp ./target/code-nav-mp-server-0.0.1.jar ./

名称太长，重命名
mv code-nav-mp-server-0.0.1.jar code-nav.jar

执行jar包
java -jar code-nav.jar

用后台启动程序
nohup java -jar code-nav.jar & 

后台启动怎么知道程序跑起来没，用jobs命令看看有没任务在跑
jobs

查看是否有java进程
ps -ef|grep 'java'

查看占用的端口
netstat -ntlp

根据端口访问一下，如
curl localhost:8082/xxx

报错就看日志
cd logs
cat error.log
查看最新10条
tail -n 10 error.log
用sz把日志下载到本地
sz error.log
确定后用vim编辑器修改代码

用kill杀死老进程
kill -9 %1

重新构建执行
nohup java -jar code-nav.jar & 

用top命令查看进程状态
top


写shell脚本
vim start.sh

内容下面三行：
./apache-maven-3.8.2/bin/mvn install
nohup java -jar code-nav.jar & 
echo "success"

执行
./start.sh

没有权限的话给该文件加可执行权限
chmod a+x start.sh

重新执行
./start.sh

ll -lh
