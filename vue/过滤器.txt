过滤器函数最好返回有意义的值；

管道源码解析
获取过滤器和参数后，整理输出为对象
{
    expression: 'a+1',
    filters: [
        {
            name: 'uppercase', args: null
        }
    ]
}
然后将表达式值作为参数，依次调用filters数组中过滤器函数，最后输出结果；

采用Vue.filter需要在实例化Vue前定义，否则不起作用；
