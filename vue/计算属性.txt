计算属性getter不执行的场景
当包含计算属性的节点被移除且模版中其他地方没有再引用该属性时，getter不会执行
<p v-if="showTotal">Total Price = {{ totalPrice }}</p>
computed: {
    totalPrice: function(){
        // basePrice变化，但是showTotal为false时这里不会执行
        return this.basePrice;
    }
}
